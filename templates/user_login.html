Login with Lightning

Scan this QR code with your Lightning wallet to log in. Remember which wallet you used as this will act as your password.

The Login with Lightning button is powered by LNURL-auth

LNURL-auth is a lightweight authentication protocol that is easy to use, implement, and respects the user's privacy.

What is LNURL-auth?

LNURL-auth is a generic authentication protocol using digital signatures. This means users only need a key pair. With the growing popularity of lightning wallets, users easily have such keys. No other identifying information (passwords, emails, usernames) is required.

How does it work?

The seed of your Bitcoin wallet generates secret authentication keys. When you log in to a site, the site poses a challenge only solvable by your secret key. A more technical explanation can be found here.

Are identities persistent?

Yes, but they don't transfer across sites or wallets. Each domain uses a different key, so no cross-domain association is possible. As long as you use the same wallet for the same site, your identity remains consistent.
